# Quiz-app-using-java
🎯 Ultimate Quiz App

A modern and interactive Quiz Application built using HTML, CSS, and JavaScript.  
This project includes category-based quizzes, animated effects, sound effects, profile system, score tracking, and a beautiful 3D animated background using Three.js.

---

## 🚀 Features

- 🔐 Login Screen
- 👤 Avatar Selection
- 📚 Multiple Categories
  - General Knowledge
  - Tech
  - Aptitude
- 🎮 Randomized Questions
- ✅ Live Score Tracking
- 🔊 Sound Effects
- ✨ Animations for Correct/Wrong Answers
- 🎉 Confetti Celebration
- 🌌 3D Animated Background
- 📱 Responsive UI Design
- 💾 Local Storage Support

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Three.js
- Canvas Confetti
